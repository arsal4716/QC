### Medicare

**Qualified:**

* Age ≥ 65 (or disabled with proper SSDI history per campaign rules)
* US citizen or permanent resident (5+ years)
* Has active Medicare Part A & B
* Has Medicare card (red, white, blue)
* Valid SSN (if required)
* Not on full Medicaid (if campaign disallows)
* Not in disallowed Medicare Advantage situations

**Not Qualified:**

* Age < 65 and not covered by disability rules
* No Medicare Part A/B or no Medicare card
* In nursing home / long-term care when campaign excludes this
* Has VA/Tricare/employer coverage that conflicts
* Non-US resident
* Medicaid only, no Medicare
* No SSN when required
* Refuses to verify coverage when required

**Map Medicare NQ outcomes as:**

* Nursing home / long-term care → `Not Qualified → CX in Nursing Home`
* Medicaid only → `Not Qualified → CX has Medicaid only`
* VA/Tricare/employer/residency/SSN issues → `Not Qualified → CX Condition Not Met` or `CX Disqualifies`

*Never use* `Not Qualified → CX has Medicare` *inside Medicare campaigns.*

When the active campaign is Medicare:

1. If CX hangs up at any point →
   Disposition: Not Interested → CX hangup Call

2. If CX states they are only looking for money, cash benefits, or subsidy →
   Disposition: Not Interested → CX not interested

3. If CX has ONLY Part A or ONLY Part B →
   Disposition: Not Qualified → CX Condition Not Met

4. If CX does NOT have BOTH Part A AND Part B →
   Disposition: Not Qualified → CX Condition Not Met

5. If CX already has Medicare AND refuses to change plans →
   Disposition: Not Interested → CX happy with current plan

IMPORTANT:

* Medicare status is INPUT only, never a disposition in Medicare campaigns.
* Do NOT use "CX has Medicare" in this campaign.
